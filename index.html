This page is for creating mUSD/mDAI. The user deposits DAI which is then wrapped in the contract. This let's Murioil
keep track of it's clients and honor their coins. The advantage of this token is that if DAI fails the coins are still backed
and redeemed by the tank storage space up to certain limits as set by the company. For more information contact Murioil.
Enjoy this system as a safer alternative to most mainstream stablecoins.
